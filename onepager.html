AGENTS.md とは？／全体像

AGENTS.md は、ひとことで言うと 「AIエージェント向けのREADME」。プロジェクト直下に置くと、対応AIが作業前に自動で読み、手順・規約・禁止事項を把握します。中身はただのMarkdownで、必須項目はありません。比喩でいえば「新メンバーへの引き継ぎメモ」です。✅

## コード規約 — 2スペース・camelCase ／ 依存は勝手に足さない
## やらないこと — /legacy は変更しない ／ .env はコミットしない

Claude Code でどう使うか・活用

Claude Code は AGENTS.md を“そのまま”は読みません（読むのは CLAUDE.md）。橋渡しが必要です。✅

import（推奨）

CLAUDE.md の中で @AGENTS.md を読み込む。

ln -s AGENTS.md CLAUDE.md でリンク。

  • 最小主義：コマンド・禁止事項・役割に絞る ✅🔶
